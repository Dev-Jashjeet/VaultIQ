Personal Finance Manager is a web-based application that helps users efficiently track their income, expenses, and overall financial activity. It allows users to securely sign up/login, manage transactions, and view their financial data in an organized dashboard. The project is built using HTML, CSS, and TypeScript, with local storage used for managing user data and transactions, focusing on clean UI and strong frontend logic implementation.
